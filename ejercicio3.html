La Importancia de la Semántica

Usar la etiqueta correcta garantiza que todos los usuarios puedan interactuar con la página, independientemente de cómo naveguen.

Ejemplo Incorrecto (Antipatrón)

Si usamos un enlace para ejecutar una acción en la misma página, los lectores de pantalla anuncian "Enlace", lo que confunde al usuario. Además, no se puede activar con la barra espaciadora.

Ejemplo Correcto (Semántico)

Al usar la etiqueta nativa, el lector de pantalla anuncia "Botón" y el navegador ya sabe que debe responder tanto a la tecla Enter como a la barra Espaciadora.

¿Qué normas WCAG respalda esto?

  • WCAG 4.1.2 (Nombre, Función, Valor): Exige que el software sepa qué hace cada elemento. Usar <button> le da el rol de "botón" automáticamente.
  • WCAG 2.1.1 (Teclado): Exige que todo se pueda usar con teclado. El <button> nativo cumple esto sin tener que programar nada extra.
